Documentation

The myFlix API is a REST API. It can be accessed via HTTP methods like GET and POST. MyFlix Api has information on movies, directors, and genres

Business Logic URL HTTP Methods Request Body data format Response Body data format
Return a list of ALL movies to the user /movies GET None A JSON object containing name and image of every movie in the database
Return movie data by movie title /movies/[Title] GET None A JSON object containing details of one specific movie
Return data about a genres by genre title /genres/[Title] GET None A JSON object containing a description of a specific genre
Return data about a director by name /directors/[Name] GET None A JSON object containing biographical details of one specific director
Allow new users to register /users POST A JSON object containing user data A Text message indicating that the registration process was successful
Allow users to update their username /users/[Name] PUT None A Text message indicating that the update was successful
Allow users to add a movie to their list of favorites /favorites/[Title] PUT None A Text message indicating that the update was successful
Allow users to remove a movie from their list of favorites /favorites/[Title] DELETE None A Text message indicating that the deletion was successful
Allow existing users to deregister /users/[Name] DELETE None A Text message indicating that the deletion was successful

USers - Allow user to register

POST /users

Example request:

        {
            "username" : "defenderNo1",
            "name" : "Jessica Jones",
            "eMail" : "jessica-jones@domain.com"
            "password" : "########"
        }

Example response:

        You're successfully regitered.'

Users - Allow user to update username

PUT /users/marvelheroNo1

Example response:

        Your username has successfully been updated.

Users - Allow user to delete account

DELETE /users/defenderNo1

Example response:

        Your username has successfully been deleted.

Favorites - Allow user to add movies to favorites list

PUT /favorites/Young%20Adult

Example response:

        Your movie has successfully been added to favorites.

Favorites - Allow user to delete movies from favorites list

DELETE /favorites/Young%20Adult

Example response:

        Your movie has successfully been deleted from favorites.
