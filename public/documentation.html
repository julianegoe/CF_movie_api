Documentation

The myFlix API is a REST API. It can be accessed via HTTP methods like GET and POST. MyFlix Api has information on movies, directors, and genres.

Business Logic URL HTTP Methods Request Body data format Response Body data format
Return a list of ALL movies to the user /movies GET None A JSON object containing name and image of every movie in the database
Return movie data by movie title /movies/[Title] GET None A JSON object containing details of one specific movie
Return data about a genres by genre title /genres/[Title] GET None A JSON object containing a description of a specific genre
Return data about a director by name /directors/[Name] GET None A JSON object containing biographical details of one specific director
Allow new users to register /users POST A JSON object containing user data A JSON object containing user data
Allow users to update user info /users/[Name] PUT A JSON object containing "Username", "Password", "Email", "Birthday" A JSON object containing date of one user
Allow users to add a movie to their list of favorites /users/[username]/movies/[movieId] PUT None A JSON object containing user data including updated movieId
Allow users to remove a movie from their list of favorites /users/[username]/movies/[movieId] DELETE None A Text message indicating that the deletion was successful
Allow existing users to deregister /users/[Name] DELETE None A Text message indicating that the deletion was successful
Return a list of ALL users to the user /users GET None A JSON object containing date of all users
